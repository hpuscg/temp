### 升级程序的启动方法

* Windows下直接双击运行upgrade4highAlarm.exe
* linux系统下，加权限后直接./upgrade4highAlarm_linux或者./upgrade4highalarm_mac

### 配置文件说明

* config.txt是限高报警升级工具的配置文件
* 内容格式为：ip=真实IP（请勿修改格式，以免造成程序执行失败）
* 真实IP即为LibraT的真实IP
